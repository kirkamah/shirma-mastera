# Сборка версии для macOS

«Ширма Мастера» — это Electron-приложение, поэтому под macOS оно работает,
но **собрать .dmg можно только на macOS** (нативный модуль базы `better-sqlite3`
и упаковщик `.dmg` не собираются на Windows). Ниже — два пути.

Целевая система: **macOS 11 Big Sur и новее** (у получателя 11.7.10 — подходит).
Сборка **универсальная** — один файл работает и на Intel, и на Apple Silicon.
Приложение **без подписи Apple**, поэтому при первом запуске его открывают через
правый клик → «Открыть» (см. ниже).

Иконка macOS лежит в `build/icon.icns` (генерируется `npm run icon:mac`).

---

## Путь A — облачная сборка через GitHub Actions (без своего Mac)

1. Создай репозиторий на GitHub и залей туда проект:
   ```bash
   cd C:\projects\shirma-mastera
   git init
   git add .
   git commit -m "Ширма Мастера"
   git branch -M main
   git remote add origin https://github.com/<твой-аккаунт>/shirma-mastera.git
   git push -u origin main
   ```
   > `node_modules`, `out`, `release` пушить не нужно — добавь их в `.gitignore`,
   > если его ещё нет.
2. На GitHub открой вкладку **Actions** → workflow **«Build macOS app»** →
   **Run workflow**.
3. Через ~5–10 минут открой завершившийся запуск и скачай артефакт
   **`shirma-mastera-mac-dmg`** — внутри `.dmg`.
4. Передай `.dmg` товарищу.

(Workflow уже лежит в `.github/workflows/build-mac.yml`.)

---

## Путь B — сборка на «живом» Mac

На Маке (с установленным Node 20+):
```bash
git clone <repo>           # или просто скопируй папку проекта на Mac
cd shirma-mastera
npm install
npm run build:mac
```
Готовый файл появится в `release/Ширма Мастера-0.1.0-mac-universal.dmg`.

## Как товарищу открыть неподписанное приложение

1. Открыть `.dmg`, перетащить «Ширма Мастера» в «Программы».
2. Запускать **не двойным кликом**, а: правый клик по иконке → **«Открыть»** →
   в окне предупреждения снова **«Открыть»**. Дальше запускается как обычно.
3. Если macOS всё равно ругается «повреждено/не удаётся проверить», в Терминале:
   ```bash
   xattr -dr com.apple.quarantine "/Applications/Ширма Мастера.app"
   ```

## Заметки

- Подпись и нотаризация Apple (чтобы открывалось двойным кликом без предупреждений)
  требуют аккаунт Apple Developer ($99/год). Для передачи другу это не обязательно.
- Данные приложения (база, заметки) на macOS хранятся в
  `~/Library/Application Support/Ширма Мастера` — переносятся отдельно от .dmg.
- Функционально всё кроссплатформенно (база, Open5e API, экспорт PDF), особой
  адаптации кода под macOS не требуется.
